canteenOnline
=============
A small project for a hackathon.

A full online interface to manage online booking of food in a restaurant.

Features
- Interface for restaurant managener.
  - Add available item
  - Change prices
  - Set veg nonveg
  - Add admin user
  - See orders
  - Set orders as seen or served
- Interface for user
  - Place order
  - See older orders
  - Create new user
